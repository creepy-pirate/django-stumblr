# Stumblr

Stumblr Application is Social Application similar to discord with chatrooms and more.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dependencies.

```bash
pip install Django
```

## Usage

```bash

# start the server 
python3 manage.py runserver
```
